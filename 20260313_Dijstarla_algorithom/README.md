## 🎯 算法核心

Dijkstra 算法是一种贪心算法，用于求解单源最短路径问题。

**核心思想：**
```
每次从未访问的节点中选择距离起点最近的节点，
更新其邻接节点的距离。
```

**核心公式：**
```
dist[v] = min(dist[v], dist[u] + weight(u, v))
```

## 📊 算法复杂度

### 时间复杂度

- **数组实现**：O(V²)，其中 V 是节点数量
- **优先队列（最小堆）**：O(E + V log V)，其中 E 是边数
- **斐波那契堆**：O(E + V log V)

### 空间复杂度

- O(V) - 存储距离、访问标记和前驱节点
- O(E) - 存储图的邻接表

## 💡 重要提示

- Dijkstra 算法只适用于**非负权重**的图
- 如果存在负权边，需要使用 Bellman-Ford 算法
- 算法使用贪心策略，每次选择最优的局部解
- 优先队列优化可以显著提高性能
