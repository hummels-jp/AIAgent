拓扑连通图概述 / トポロジカル接続グラフの概要 / Topological Connectivity Graph Overview

拓扑连通图 (Topological Connectivity Graph)是机器人路径规划中用于表示环境中自由空间连接关系的数据结构。它将连续的工作空间离散化为图结构,其中节点表示可达位置,边表示可达路径。

Topology Connectivity Graph (トポロジカル接続グラフ) is a data structure used in robot path planning to represent the connectivity of free space in the environment. It discretizes the continuous workspace into a graph structure, where nodes represent reachable positions and edges represent reachable paths.

トポロジカル接続グラフは、ロボット経路計画において環境内の自由空間の接続関係を表すデータ構造です。連続する作業空間をグラフ構造に離散化し、ノードは到達可能な位置を、エッジは到達可能なパスを表します。

主要特点 / 主な特徴 / Key Features:

  • 拓扑结构 (Topology / トポロジー)：只关心位置之间的连接关系，不关心具体几何形状 / 位置間の接続関係のみを考慮し、具体的な幾何形状は考慮しません / Only concerns connectivity between positions, not specific geometric shapes
  • 连通性 (Connectivity / 接続性)：确保图中任意两个可达节点之间存在路径 / グラフ内の任意の到達可能ノード間にパスが存在することを保証 / Ensures there is a path between any two reachable nodes in the graph
  • 完整性 (Completeness / 完全性)：能够表示环境中的所有可行路径 / 環境内のすべての実行可能なパスを表現できます / Can represent all feasible paths in the environment
  • 简洁性 (Simplicity / 簡潔性)：相比网格表示，大大减少状态空间 / グリッド表現と比較して状態空間を大幅に削減 / Greatly reduces state space compared to grid representation

应用场景 / 応用シーン / Application Scenarios:

  • 室内环境导航（家庭、办公室、仓库） / 室内環境ナビゲーション（家庭、オフィス、倉庫） / Indoor environment navigation (home, office, warehouse)
  • 室外路径规划（城市道路、野外环境） / 屋外経路計画（都市道路、野外環境） / Outdoor path planning (urban roads, wilderness environments)
  • 多机器人协同路径规划 / マルチロボット協調経路計画 / Multi-robot collaborative path planning
  • 动态环境中的实时路径更新 / 動的環境におけるリアルタイム経路更新 / Real-time path updates in dynamic environments

可视图 / 可視グラフ / Visibility Graph

可视图 (Visibility Graph)是一种基于可见性约束的拓扑图构建方法。它将障碍物的顶点作为图的节点，并在所有相互可见的顶点之间建立边连接。

Visibility Graph is a topology graph construction method based on visibility constraints. It takes obstacle vertices as graph nodes and establishes edge connections between all mutually visible vertices.

可視グラフは、可視性制約に基づくトポロジーグラフ構築方法です。障害物の頂点をグラフのノードとし、相互に可視な頂点間でエッジ接続を確立します。

核心思想 / 核心的アイデア / Core Concept:

如果两点之间的连线不与任何障碍物相交，则这两点在可视图中是"可见"的，可以建立连接。这样构建的图中的最短路径就是实际环境中的最短路径。

If the line segment between two points does not intersect any obstacle, then these two points are "visible" in the visibility graph and a connection can be established. The shortest path in the constructed graph is the actual shortest path in the environment.

2点間の線分がいかなる障害物とも交差しない場合、これら2点は可視グラフで「可視」とみなされ、接続を確立できます。このように構築されたグラフの最短経路は、実際の環境内の最短経路です。

可见性判断条件 / 可視性判断条件 / Visibility Judgment Condition:
点P1(x1, y1)和点P2(x2, y2)可见，当且仅当：
点P1(x1, y1)と点P2(x2, y2)が可視である場合、かつその場合に限り：
Points P1(x1, y1) and P2(x2, y2) are visible if and only if:
∀ 障碍物边E 障害物辺E obstacle edge E: segment(P1, P2) ∩ E = ∅

步骤2：添加起点和终点 / ステップ2：起点と終点の追加 / Step 2: Add Start and End Points

对每对顶点，判断它们之间的线段是否与任何障碍物相交 / 頂点の各ペアについて、それら間の線分がいかなる障害物とも交差するかを判断 / For each pair of vertices, determine whether the line segment between them intersects any obstacle

Voronoi图 / ボロノイ図 / Voronoi Diagram

Voronoi图 (ボロノイ図)是一种基于距离的划分方法，将平面划分为若干区域，每个区域包含所有距离某个种子点最近的点。在路径规划中，Voronoi图的边线是距离障碍物最远的安全路径。

Voronoi Diagram is a distance-based partitioning method that divides the plane into multiple regions, each containing all points closest to a specific seed point. In path planning, the edges of the Voronoi diagram represent the safest paths that are farthest from obstacles.

ボロノイ図は、距離に基づく分割方法で、平面を複数の領域に分割し、各領域は特定のシード点に最も近い点をすべて含みます。経路計画では、ボロノイ図の辺は障害物から最も遠い安全な経路を表します。

核心思想： / 核心的アイデア： / Core Concept:

Voronoi图由所有到两个或多个障碍物等距的点组成。这些点形成了一条"脊线"，在机器人导航中代表最安全的路径，因为它们距离所有障碍物的距离都相等且最大。

Voronoi定义： / Voronoi定義： / Voronoi Definition:
对于障碍物集合O，点P属于障碍物O_i的Voronoi区域，当且仅当：
distance(P, O_i) ≤ distance(P, O_j) for all j ≠ i
Voronoi边（脊线）： / Voronoi辺（脊線）： / Voronoi Edge (Ridge Line):
点P位于Voronoi边上，当且仅当：
∃ i, j: distance(P, O_i) = distance(P, O_j) < distance(P, O_k) for all k ≠ i,j

构建方法： / 構築方法： / Construction Methods:

方法1：离散点法 / 方法1：離散点法 / Method 1: Discrete Point Method
1. 在环境中均匀采样大量离散点
2. 对每个点，计算到所有障碍物的距离
3. 找到最近和次近的障碍物
4. 如果距离相等（在误差范围内），则该点在Voronoi边上
方法2：增量法（适用于少量种子点） / 方法2：増分法（少数のシード点に適用） / Method 2: Incremental Method (for few seed points)
1. 从两个种子点开始，构建它们的垂直平分线
2. 逐个添加新种子点，更新Voronoi图
3. 处理新增顶点和边的关系
4. 删除被覆盖的区域
方法3：波前扩散法（适用于复杂环境） / 方法3：波前拡散法（複雑な環境に適用） / Method 3: Wavefront Propagation (for complex environments)
1. 从所有障碍物边界同时开始扩散
2. 两个波前相遇的位置就是Voronoi边
3. 三个或多个波前相遇的位置是Voronoi顶点
4. 类似于计算距离变换和提取骨架

优点： / 長所： / Advantages:

  • ✓ 路径距离障碍物最远，安全性高
  • ✓ 对障碍物形状不敏感，适用性广
  • ✓ 天然提供了多条备选路径
  • ✓ 适用于凸形和凹形障碍物

缺点： / 短所： / Disadvantages:

  • ✗ 路径通常不是最短路径
  • ✗ 构建计算复杂度较高
  • ✗ 在狭窄通道中可能不稳定
  • ✗ 生成的路径可能过于曲折
