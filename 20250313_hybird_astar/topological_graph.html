拓扑连通图构建方法

可视图与Voronoi图详解

拓扑连通图概述

拓扑连通图是机器人路径规划中用于表示环境中自由空间连接关系的数据结构。它将连续的工作空间离散化为图结构，其中节点表示可达位置，边表示可达路径。

主要特点：

  • 拓扑结构：只关心位置之间的连接关系，不关心具体几何形状
  • 连通性：确保图中任意两个可达节点之间存在路径
  • 完整性：能够表示环境中的所有可行路径
  • 简洁性：相比网格表示，大大减少状态空间

应用场景：

  • 室内环境导航（家庭、办公室、仓库）
  • 室外路径规划（城市道路、野外环境）
  • 多机器人协同路径规划
  • 动态环境中的实时路径更新

可视图

可视图是一种基于可见性约束的拓扑图构建方法。它将障碍物的顶点作为图的节点，并在所有相互可见的顶点之间建立边连接。

核心思想：

如果两点之间的连线不与任何障碍物相交，则这两点在可视图中是"可见"的，可以建立连接。这样构建的图中的最短路径就是实际环境中的最短路径。

可见性判断条件：
点P1(x1, y1)和点P2(x2, y2)可见，当且仅当：
∀ 障碍物边E: segment(P1, P2) ∩ E = ∅

构建算法步骤：

步骤1：提取障碍物顶点
收集所有障碍物的所有顶点，作为图的候选节点
步骤2：添加起点和终点
将起点和终点作为额外的节点加入图中
步骤3：判断可见性
对每对顶点，判断它们之间的线段是否与任何障碍物相交
步骤4：建立边连接
如果两点相互可见，则在它们之间建立边，边的权重为欧几里得距离
步骤5：路径搜索
使用A*等图搜索算法在构建的可视图中寻找从起点到终点的最短路径

优点：

  • ✓ 能够找到理论上的最短路径
  • ✓ 图的规模相对较小，计算效率高
  • ✓ 适用于多边形障碍物环境

缺点：

  • ✗ 路径可能过于贴近障碍物，不安全
  • ✗ 需要精确的障碍物几何信息
  • ✗ 不考虑运动学约束

Voronoi图

Voronoi图是一种基于距离的划分方法，将平面划分为若干区域，每个区域包含所有距离某个种子点最近的点。在路径规划中，Voronoi图的边线是距离障碍物最远的安全路径。

核心思想：

Voronoi图由所有到两个或多个障碍物等距的点组成。这些点形成了一条"脊线"，在机器人导航中代表最安全的路径，因为它们距离所有障碍物的距离都相等且最大。

Voronoi定义：
对于障碍物集合O，点P属于障碍物O_i的Voronoi区域，当且仅当：
distance(P, O_i) ≤ distance(P, O_j) for all j ≠ i
Voronoi边（脊线）：
点P位于Voronoi边上，当且仅当：
∃ i, j: distance(P, O_i) = distance(P, O_j) < distance(P, O_k) for all k ≠ i,j

构建方法：

方法1：离散点法
1. 在环境中均匀采样大量离散点
2. 对每个点，计算到所有障碍物的距离
3. 找到最近和次近的障碍物
4. 如果距离相等（在误差范围内），则该点在Voronoi边上
方法2：增量法（适用于少量种子点）
1. 从两个种子点开始，构建它们的垂直平分线
2. 逐个添加新种子点，更新Voronoi图
3. 处理新增顶点和边的关系
4. 删除被覆盖的区域
方法3：波前扩散法（适用于复杂环境）
1. 从所有障碍物边界同时开始扩散
2. 两个波前相遇的位置就是Voronoi边
3. 三个或多个波前相遇的位置是Voronoi顶点
4. 类似于计算距离变换和提取骨架

优点：

  • ✓ 路径距离障碍物最远，安全性高
  • ✓ 对障碍物形状不敏感，适用性广
  • ✓ 天然提供了多条备选路径
  • ✓ 适用于凸形和凹形障碍物

缺点：

  • ✗ 路径通常不是最短路径
  • ✗ 构建计算复杂度较高
  • ✗ 在狭窄通道中可能不稳定
  • ✗ 生成的路径可能过于曲折

可视图 vs Voronoi图 对比

特性 可视图 Voronoi图
路径类型 最短路径 安全路径（最大间隙）
与障碍物距离 可能非常近 保持最大距离
计算复杂度 O(n²) O(n log n) ~ O(n²)
障碍物约束 精确几何信息 只需要距离信息
适用环境 多边形障碍物 任意形状障碍物
路径质量 最优（距离） 次优（安全性高）
实现难度 中等 较高
备选路径 较少 较多

选择建议：

  • 使用可视图：当需要最短路径、障碍物是精确多边形、计算资源充足时
  • 使用Voronoi图：当需要安全路径、障碍物形状复杂、环境不确定时
  • 混合方法：先用Voronoi图规划安全走廊，再在走廊内用可视图优化路径

扩展应用

除了可视图和Voronoi图，还有其他拓扑图构建方法，每种都有其适用场景。

栅格图

简单但计算量大

PRM路标图

适用于高维空间

RRT快速探索树

适用于复杂动态环境

其他方法简介：

  • 栅格图：将环境划分为均匀网格，适合简单环境，但状态空间大
  • PRM：概率路标图，随机采样构建图，适用于高维配置空间
  • RRT：快速探索随机树，通过随机采样快速构建搜索树
  • Theta*：基于可视图的改进，允许路径任意穿越网格
  • 导航网格：将自由空间分割为凸多边形，用于游戏AI
