## 🎯 算法核心

A* 算法的核心公式：

```
f(n) = g(n) + h(n)
```

其中：
- **f(n)** - 节点 n 的总评估值
- **g(n)** - 从起点到节点 n 的实际代价
- **h(n)** - 从节点 n 到终点的估计代价（启发式函数）

启发式函数的选择：
- 曼哈顿距离：适用于只能上下左右移动的网格
- 欧几里得距离：适用于可任意方向移动的连续空间
- 切比雪夫距离：适用于可对角线移动的网格

## 💡 提示

- 启发式函数 h(n) 必须满足可采纳性（不超过实际最小代价）
- 启发式函数越接近实际代价，算法效率越高
- 当 h(n) = 0 时，A* 退化为 Dijkstra 算法
- 当 g(n) = 0 时，A* 退化为贪心最佳优先搜索
